# Sistema de Gestão de Doações - Associação NutriAção
Sistema desenvolvido para auxiliar a ONG Associação NutriAção no gerenciamento de:

- Doações
- Famílias beneficiadas
- Entregas
- Relatórios
- Controle de validade dos alimentos

## Tecnologias
- React.js
- Node.js
- Express
- MySQL
